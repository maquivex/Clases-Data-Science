| Información a extraer             | Tag                               | Clase / Atributo                                                         | Cómo extraerlo con BeautifulSoup                                                           |                                             |
| --------------------------------- | --------------------------------- | ------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------ | ------------------------------------------- |
| **Contenedor del anuncio**        | `article`                         | `class="c-results c-results-used--premium"`                              | `soup.find_all('article', class_='c-results')`                                             |                                             |
| **Enlace del auto**               | `a`                               | `class="c-results__link"` y `href`                                       | `art.find('a', class_='c-results__link')['href']`                                          |                                             |
| **Título / Modelo / Año**         | `h2`                              | `class="c-results__header-title"`                                        | `art.find('h2', class_='c-results__header-title').text.strip()`                            |                                             |
| **Etiqueta “Seminuevo”**          | `div`                             | `class="c-results-tag__stick"`                                           | `art.find('div', class_='c-results-tag__stick').text.strip()`                              |                                             |
| **Imagen principal**              | `img`                             | `class="c-results-slider__img-inside"` y `data-src`                      | `art.find('img', class_='c-results-slider__img-inside')['data-src']`                       |                                             |
| **Combustible**                   | `span`                            | `class="c-results-used__detail-fuel"`                                    | `art.find('span', class_='c-results-used__detail-fuel').text.strip()`                      |                                             |
| **Transmisión**                   | `p` o `span`                      | siguiente al anterior (`                                                 | Automática`)                                                                               | extraer texto completo del `<p>` contenedor |
| **Kilometraje**                   | `p`                               | `class="c-results-details__description-text"` que contiene “kilometraje” | buscar por texto: `art.find('p', string=lambda t: 'kilometraje' in t.lower())`             |                                             |
| **Ubicación (Ciudad, Provincia)** | `span`                            | `class="c-results-details__description-text--highlighted"`               | `art.find('span', class_='c-results-details__description-text--highlighted').text.strip()` |                                             |
| **Precio Santander**              | `div`                             | `class="c-results-mount__santander-price"`                               | `art.find('div', class_='c-results-mount__santander-price').text.strip()`                  |                                             |
| **Precio normal**                 | `div`                             | `class="c-results-mount__price"`                                         | `art.find('div', class_='c-results-mount__price').text.strip()`                            |                                             |
| **Nombre del anunciante**         | `data-gtm` dentro del `<article>` | clave `"item_advertiser"`                                                | usar `json.loads(article['data-gtm'])['item_advertiser']`                                  |                                             |
| **Marca**                         | dentro de `data-gtm`              | `"item_brand"`                                                           | igual que arriba                                                                           |                                             |
| **Año**                           | dentro de `data-gtm`              | `"item_year"`                                                            | igual que arriba                                                                           |                                             |
| **Tipo de combustible**           | dentro de `data-gtm`              | `"item_fuel"`                                                            | igual que arriba                                                                           |                                             |
| **Ubicación (ciudad)**            | dentro de `data-gtm`              | `"item_location_city"`                                                   | igual que arriba                                                                           |                                             |
